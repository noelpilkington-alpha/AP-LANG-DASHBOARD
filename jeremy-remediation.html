Phase 1: Building the Argument-Tracing Skill

1 Coaching Call | ~8-10 hours independent work | Week 1

Module 1: Diagnostic

Confirm the hypothesis: Jeremy can build arguments but can't trace them.

Part 1: Writing-to-Reading Test

  • Give Jeremy a topic → he outlines an Argument essay (thesis, paragraph functions, evidence types, counterargument placement)
  • Then give him a published passage with SIMILAR argument structure → identify thesis, paragraph functions, counterargument
  • Compare: does the author's structure match what he planned?

Part 2: Style vs. Structure Comparison

Same passage, two annotations: style choices (strength) vs. argument structure (weakness). Is the style annotation richer and more confident? This reveals where his reading attention goes.

Part 3: MCQ Baseline

10 MCQs, mix of all skills, timed at exam pace.

Module 2: How Arguments Are Built (The Writing-to-Reading Bridge)

Jeremy-Specific Module: Sara Beth's Module 2 taught argument structure from scratch. Jeremy's uses his EXISTING writing knowledge as the bridge. The innovation: show Jeremy that published passages use the same moves he uses in his own essays.

Skill Targets

Skill 5: REO Reading10% → 50%+ (5x improvement)
Topics 3.1, 3.3, 5.2~0% → 50-60%+

Step 1: The Bridge Exercise (Coaching Call #1, 45 min)

  1. Ask Jeremy: "How would you structure an argument essay about [topic]?" He produces paragraph-by-paragraph functions.
  2. Give him a published passage on the SAME topic. Walk through together:
    "P1: The author does exactly what you planned — establishes the problem and states a thesis. But notice HOW: the thesis is embedded in the last sentence."
    "You said you'd use a 'different angle' for P3. This author uses COMPARISON-CONTRAST instead of example. Same FUNCTION, different STRUCTURE."
  3. Draw both argument maps side by side. Show the structural correspondence.
  4. Do a SECOND passage with a DIFFERENT structure (inductive vs. deductive). Compare again.

Step 2: Guided Practice (~3 hours)

3 passages with decreasing scaffolding:

  • Passage A: Jeremy outlines first, then reads and maps. Guided comparison.
  • Passage B: Jeremy reads first, maps structure, THEN writes how he'd structure it differently.
  • Passage C: Maps structure only (no personal outline). Full argument map.

Steps 3-4: Independent Practice + Assessment

3 new passages (~800-1000 words), full argument maps, no scaffolding. Then 10 Skill 5 MCQs, timed, single attempt, explain-your-reasoning.

Exit Criteria

  • 50%+ on Skill 5 assessment (5x improvement from 10%)
  • Function-level language: "this paragraph provides a counterexample that qualifies the thesis"
  • Not content-level: "this paragraph is about schools"
  • Jeremy identifies: "I read for CONTENT when I should have read for STRUCTURE"

Module 3: Claims in Reading (Jeremy-Specific)

Jeremy-Specific Module: Skill 3 (CLE Reading): 20%. Skill 4 (CLE Writing): 71%. Same reading/writing split. He can PRODUCE claims and evidence but can't reliably IDENTIFY them in reading.

Skill Targets

Skill 3: CLE Reading20% → 50%+
Skill 4: CLE Writing71% → 71%+ (maintain)
Course: Relating to Audience58.3% → 75%+
Course: Flaws in Reasoning68% → 80%+

Activities

  • Worked Example: Annotate Jeremy's OWN practice essay for claims/evidence, then find the SAME structures in a published passage
  • Distinguish: Thesis (overarching) vs. Claim (paragraph-level) vs. Evidence (support)
  • Claim identification drill: 5 passages, identify thesis + all paragraph claims + evidence types
  • Audience analysis: 2 passages, same topic, different audiences — how evidence changes
  • Flaws in reasoning: 3 passages with logical fallacies
  • Assessment: 15 MCQs (10 Skills 3/4 + 5 cumulative Skill 5)

Exit Criteria

  • 50%+ on Skills 3/4 AND 50%+ maintained on Skill 5
  • Can distinguish thesis, claims, and evidence in unseen passages
  • Writing MCQs (Skill 4): maintained at 71%+

Module 6: Targeted Repair + Essay Intensive

Targeted MCQ Repair (~3 hours)

  • If Skill 5 still below 40%: Return to Module 2 writing-to-reading bridge with new passages
  • If Skill 3 still below 40%: Return to Module 3 claims-in-reading drill
  • If Skill 8 still at 0%: Return to Module 4 worked examples

Essay Intensive (~3 hours)

  • Rhetorical Analysis (40 min): Push 4/6 → 5/6. STRUCTURE → FUNCTION → EFFECT as outline.
  • Synthesis (40 min): Push 4/6 → 5/6. Source integration + counterargument with concession.
  • Sophistication practice: Revise one paragraph using qualification, broader context, or alternative interpretation.

Exit Criteria

  • Targeted skills improving toward 50%+
  • Each essay scored 5/6 or above
  • Sophistication point earned on 1+ of 2 essays

Phase 3: Final Readiness

1 Coaching Call | ~4-5 hours independent work | Week 3-4

Module 7: Final Simulation + Exam Strategy

Final Simulation

45 MCQs (45 min) + 3 FRQs (2 hrs 15 min). Different material from Module 5. Full exam conditions.

Exam Strategy (Coaching Call #2, 30-45 min)

  • MCQ: Leverage style strength (Skill 7: 83%) — fast, confident points. Invest saved time in Skill 5 questions.
  • FRQ order: Argument (strongest, 5/6) → Synthesis → Rhetorical Analysis. 40 min each + 15 min re-reading.
  • Sophistication moves:
    • Argument: qualification of claims ("while X is generally true, in cases where Y...")
    • Synthesis: concession + rebuttal using a source ("Source E's concern about X has merit, yet...")
    • Rhetorical Analysis: broader context ("writing during the [historical moment], the author's choice to...")
  • Safety net: When lost on a structure question, ask "How would I have WRITTEN this paragraph? What would its JOB be in MY essay?"

Phase Exit Criteria

  • MCQ: 60%+ (18+ point improvement from 42%)
  • Skill 5: 55%+; Skill 3: 55%+; Skill 8: 50%+; No skill below 40%
  • FRQ: 5.0/6 average; Sophistication on 2/3+ essays
  • Transfer gap < 10 points
  • Can articulate exam-day strategy in own words
