Remediation Course: Sara Beth Way

Target: Score 2 → Score 5 | 9 Modules | ~5 weeks active + 1 week maintenance

Root Cause

Sara Beth can label rhetorical elements but cannot trace how an author builds, organizes, or deploys an argument — the single analytical skill the AP exam tests across all 8 skills and all 3 essay types.

Key behavioral pattern: +18% retake inflation (56% first-attempt → 74% all-attempts) demonstrates memorization of question-answer pairs, not genuine analytical skill. The -23% course-to-exam drop confirms skills don't transfer to novel passages.

Design Principles

The Analytical Framework: STRUCTURE → FUNCTION → EFFECT

This is the ONE move Sara Beth needs to learn. It applies identically across all skill clusters:

Phase 1: Building the Analytical Framework

1 Coaching Call | ~8-10 hours independent work | Week 1

Module 1: Diagnostic

Establish baselines. One unseen passage (~800 words): identify thesis, describe paragraph FUNCTIONS, explain evidence choice, explain style effect, identify clauses (grammar baseline).

Exit Criteria

  • If cannot identify thesis or paragraph functions: Module 2 with extra scaffolding
  • If grammar baseline weak: embed mini-lessons in Modules 2-3

Module 2: How Arguments Are Built (Worked Examples)

Trace argument structure: paragraph functions, inter-paragraph dependencies, methods of development. The core of the entire course.

Skill Targets

Skill 5: REO Reading10% → 70%+
Topics 3.1, 3.3, 5.2All at 0% → 70%+

Step 1: Worked Example (Coaching Call #1, 45 min)

Walk through a passage with ALL analytical thinking visible. Name every paragraph's JOB, method of development, transitional logic. Draw argument map. Then do a SECOND passage with different structure.

Steps 2-4: Guided → Independent → Assessment

  • 3 passages with decreasing scaffolding (thesis given → nothing given)
  • 3 passages independently, full argument maps
  • 10 Skill 5 MCQs, timed, single attempt, explain-your-reasoning

Exit Criteria

  • 70%+ on assessment; function-level (not content-level) language in maps
  • Can explain reasoning for correct AND incorrect answers

Module 3: Scaffolding Removed

Full analytical move at exam pace. Disordered-paragraph exercises. Self-assessment against Module 1 baseline.

Skill Targets

Skill 5: REO Reading70%+ maintained
Skill 6: REO Writing25% → 60%+

Exit Criteria (Phase 1 Checkpoint)

  • 70%+ on 15-question Skill 5/6 checkpoint
  • Argument maps consistently analytical; can name top 2 error patterns
  • If not met: repeat with new passages. Do NOT advance.

Phase 2: Applying the Framework Across Skills

1 Coaching Call | ~12-15 hours independent work | Weeks 2-3

Module 4: Evidence and Audience

Apply STRUCTURE → FUNCTION → EFFECT to evidence: WHAT evidence, HOW it supports the claim for THIS audience, WHY it's strategic.

Skill Targets

Skill 3: CLE Reading40% → 70%+
Skill 4: CLE Writing43% → 70%+
Course: Modes of Persuasion21% → 70%+
Course: Source Materials25% → 70%+

Activities

  • Audience comparison: 2 passages, same topic, different audiences
  • Source evaluation: 6 sources, categorize, select 3 most strategic
  • Modes of persuasion drill: 5 passages, identify appeal + evidence + effectiveness
  • Assessment: 15 MCQs (10 Skills 3/4 + 5 cumulative 5/6)

Exit Criteria

  • 70%+ on Skills 3/4 AND 70%+ maintained on Skills 5/6

Module 5: Style Through the Analytical Lens

Apply the framework to style. Grammar embedded naturally, not taught separately. First time applying all three layers to a single passage.

Skill Targets

Skill 7: STL Reading50% → 70%+
Skill 8: STL Writing0% → 60%+
Course: Diction & Syntax31% → 70%+
Course: Parenthetical Elements33% → 70%+

Activities

  • Syntax analysis: sentence length variation, parallel structure, punctuation effects
  • Tone shift mapping: identify triggers, explain how shifts advance argument
  • Style imitation: write 200 words imitating a passage's syntactic patterns
  • Assessment: 20 MCQs (10 Skills 7/8 + 10 cumulative 3-6)

Exit Criteria

  • 60%+ on Skills 7/8 AND 70%+ maintained on all prior skills

Module 6: Writing Bridge — All Three Essay Types

Deploy the framework in Sara Beth's own writing. Synthesis, Rhetorical Analysis, and Argument essays.

Skill Targets

FRQ Q1: Synthesis~3/6 → 5/6
FRQ Q2: Rhetorical Analysis~3/6 → 5/6
FRQ Q3: Argument~3/6 → 5/6

Activities

  • Worked Example (Coaching Call #2): Walk through complete Synthesis Essay with thinking aloud
  • 3 timed essays (40 min each), self-scored against AP rubric
  • Comparative analysis: 4/6 vs. 6/6 sample essays
  • Revision targeting weakest rubric row

Exit Criteria

  • All three essays 4/6+; at least one includes counterargument paragraph
  • Self-scoring within 1 point of tutor scoring

Phase 3: Exam Integration

1 Coaching Call | ~10-15 hours independent work | Weeks 4-5

Module 7: First Full Simulation + Post-Mortem

Targets

Overall MCQ33% → 55%+
Transfer gap-23 pts → < 5 pts
FRQ Average~3/6 → 4/6+

Activities

  • Full exam simulation: 45 MCQs (45 min) + 3 FRQs (2 hrs 15 min). Exam-length passages (~1200 words).
  • Post-mortem: every incorrect MCQ analyzed by skill/topic/error type. FRQs scored against rubric. Top 3 error patterns identified.

Module 8: Essay Intensive + Targeted Repair

Activities

  • Targeted MCQ repair: 10 questions per weakest 2 skills, full explain-your-reasoning
  • 2 timed essays (weakest types), score and compare to Modules 6-7
  • Sophistication practice: revise one paragraph to earn sophistication point

Module 9: Final Simulation + Exam Strategy

Activities

  • Full exam simulation (different material from Module 7)
  • Comprehensive post-mortem comparing all checkpoints
  • Coaching Call #3: MCQ strategy, FRQ strategy, sophistication strategy. Sara Beth states plan in her own words.

Phase 3 Exit Criteria

  • MCQ: 65%+ (30+ point improvement from 33%)
  • FRQ: 5/6 average with sophistication on 2/3+ essays
  • Transfer gap < 5 points; No skill below 50%
  • Can articulate personal exam-day plan
