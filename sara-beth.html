Remediation Course: 9 Modules in 3 Phases

Unlike the Emergency plans for the other three students, Sara Beth requires a ground-up custom course that teaches the analytical framework from scratch. The course uses a single repeating pattern: STRUCTURE → FUNCTION → EFFECT, applied across all skill clusters.

PhaseModulesDurationFocusCoaching Calls
Phase 1: Building the Framework 1-3 Week 1 Diagnostic, argument mapping, paragraph functions (Skill 5/6) 1
Phase 2: Applying Across Skills 4-6 Weeks 2-3 Evidence & audience (Skill 3/4), style (Skill 7/8), essay writing 1
Phase 3: Exam Integration 7-9 Weeks 4-5 Full simulations, post-mortems, targeted repair, exam strategy 1
Maintenance — Exam Week Light practice, strategy review 0

Course Design Principles

1. One framework, applied everywhereEvery module teaches STRUCTURE → FUNCTION → EFFECT applied to a different question type
2. Worked examples firstBefore any independent practice, Sara Beth sees correct analytical reasoning modeled step-by-step
3. No retakes+18% inflation proves retakes enable memorization. Every assessment is single-attempt.
4. Explain-your-reasoningFor every MCQ: why the correct answer is correct AND why each wrong answer fails
5. Novel materials onlyNever encounter the same passage twice in assessment
6. Cumulative from Module 4Every practice set mixes current and prior skills
7. Graduated passage length~800 words (Phase 1) → ~1000 words (Phase 2) → ~1200 words (Phase 3)
